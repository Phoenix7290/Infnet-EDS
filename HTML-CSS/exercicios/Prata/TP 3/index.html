CSS - Dominando a estilização de elementos

Hierarquia

A Hierarquia funciona da seguinte forma. Dentre 2 elementos que tem a mesma propriedade, prevalece o último. confira no CSS.

Herança

Isso significa que elementos filhos terão hierarquia do pai, por isso o nome Herança. Verifique o exemplo de section e este parágrafo.

Proximidade

Por sua vez, proximidade fará com que os elementos mais especificos se sobressaia dos menos especificos.
